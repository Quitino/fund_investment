# 基金投资理财入门教程

> 面向零基础程序员的系统性基金投资指南  
> 从概念到实操，读完即可上手

---

## 前言

本教程写给对金融一无所知、但有一定逻辑思维能力的程序员。金融市场本质是一套规则系统，和代码一样有结构、有逻辑、有坑。我们的目标不是成为专业投资经理，而是用普通人的资金（100元～50万），通过基金这个工具，让钱稳健地增值。

**你会学到什么：**
- 钱放在哪里、为什么、怎么放
- 基金是什么、有哪些类型、怎么挑
- 如何构建自己的投资组合
- 如何控制风险、避开常见陷阱
- 从账户开户到第一笔买入的完整实操

**阅读建议：** 按顺序读完前三章建立认知框架，再按需深入后续章节。
