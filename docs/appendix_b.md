# 附录 B：常用术语速查表

> 本速查表收录基金投资领域最常用的术语，按首字母拼音排序，共55个词条。每个词条包含中文术语、英文对照和简明解释。

---

| 术语 | 英文 | 简释 |
|------|------|------|
| A类份额 | Class A Shares | 申购时一次性收取申购费，持有期间不额外收取销售服务费；长期持有者适用 |
| B类份额 | Class B Shares | 传统分级基金中的杠杆份额，2020年后分级基金改造完成，已基本退出市场 |
| C类份额 | Class C Shares | 不收申购费，改收按年计算的销售服务费（通常0.2%~0.4%/年）；短期持有者适用 |
| ETF | Exchange Traded Fund | 交易型开放式指数基金，在证券交易所上市交易，像股票一样买卖，追踪特定指数 |
| ETF联接基金 | ETF Feeder Fund | 以场外基金形式投资ETF的基金，便于定投，手续费略高于ETF直投 |
| FOMO | Fear of Missing Out | "害怕错过"心理，指看到他人获利后产生的焦虑和冲动跟进行为 |
| FOF | Fund of Funds | 基金中的基金，以其他基金为主要投资标的；适合不善于选基的投资者 |
| QDII | Qualified Domestic Institutional Investor | 合格境内机构投资者，指获批可以将境内资金投资海外市场的基金 |
| QFII | Qualified Foreign Institutional Investor | 合格境外机构投资者，指获批可将境外资金投资中国A股市场的机构 |
| 安全边际 | Margin of Safety | 格雷厄姆提出，指买入价格相对于内在价值的折扣幅度，折扣越大越安全 |
| 摆动效应 | Seesaw Effect | 组合内不同资产往往此消彼长，通过配置负相关资产降低整体波动的现象 |
| 标准差 | Standard Deviation | 衡量基金收益率波动幅度的统计指标，数值越大，收益波动越剧烈（风险越高） |
| 波段操作 | Market Timing | 试图通过预测市场短期涨跌来买低卖高的操作策略；学术研究证明长期难以奏效 |
| 仓位 | Position | 某类资产或某只基金在整体投资组合中所占的比例 |
| 超额收益 | Alpha | 基金实际收益相对于其基准指数的超出部分；反映基金经理的主动管理能力 |
| 承诺投资期 | Lock-up Period | 某些基金设定的最短持有期限，在此期间赎回会收取较高惩罚性赎回费 |
| 持有成本 | Cost Basis | 投资者持有某只基金的平均买入价格，用于计算盈亏 |
| 赤字财政 | Fiscal Deficit | 政府支出大于收入的财政状况，通常通过发行国债弥补；影响债券市场和利率 |
| 从众效应 | Herding Effect | 羊群效应，个人在投资决策中盲目跟随多数人行为，忽略独立判断 |
| 大类资产配置 | Asset Allocation | 将资金分配于股票、债券、货币、商品、房地产等不同大类资产的决策 |
| 定投 | Systematic Investment Plan (SIP) | 定期定额投资，按固定周期（通常每月）买入固定金额基金，平摊买入成本 |
| 短债基金 | Short-term Bond Fund | 主要投资剩余期限较短（通常1年以内）债券的债券型基金，流动性好、风险低 |
| 复权净值 | Adjusted NAV | 将历史分红再投入后的累计净值，反映基金完整的收益历史；比单位净值更准确 |
| 负相关 | Negative Correlation | 两类资产价格变动方向相反（一涨一跌），配置负相关资产能降低组合波动 |
| 费率 | Expense Ratio / Fee | 投资基金所需支付的各类成本，包括申购费、赎回费、管理费、托管费等 |
| 分红 | Dividend | 基金将已实现收益按持有份额分配给投资者；分现金红利和红利再投资两种方式 |
| 封闭式基金 | Closed-end Fund | 发行后份额固定，只能在二级市场交易，不能申购赎回；存在折溢价现象 |
| 夏普比率 | Sharpe Ratio | 每承担一单位总风险所获得的超额收益（相对无风险利率）；数值越高越好 |
| 固收+ | Fixed Income Plus | 以固收类资产为底仓（通常80%以上），配置少量股票或可转债以增强收益的基金 |
| 管理费 | Management Fee | 基金管理人收取的服务费用，从基金资产中每日计提；通常0.1%~1.5%/年 |
| 换手率 | Turnover Rate | 基金在一段时间内买卖证券的频率；高换手率意味着高交易成本和主动管理程度高 |
| 回撤 | Drawdown | 基金净值从峰值到谷底的下跌幅度；"最大回撤"指历史上最大的峰谷跌幅 |
| 货币基金 | Money Market Fund | 主要投资短期货币市场工具，如国债、存单等；流动性高，风险极低，收益约1~3% |
| 基准 | Benchmark | 用于衡量基金业绩表现的参照指标，通常是某个指数或一组指数的组合 |
| 集中度 | Concentration | 基金前10大持仓标的占净资产的比例；集中度高代表风险集中，弹性也大 |
| 净值 | Net Asset Value (NAV) | 基金单位份额的价值，等于（总资产-总负债）÷ 总份额；每个交易日收盘后公布 |
| 久期 | Duration | 衡量债券价格对利率变动敏感性的指标；久期越长，利率上升时债券价格跌幅越大 |
| 开放式基金 | Open-end Fund | 可随时申购赎回的基金；相对于封闭式基金，流动性更好 |
| 可转债 | Convertible Bond | 可在一定条件下转换为股票的债券；兼具债券的保本属性和股票的上涨弹性 |
| 利率风险 | Interest Rate Risk | 市场利率变化导致债券价格波动的风险；利率上升时债券价格下跌 |
| 流动性风险 | Liquidity Risk | 无法以合理价格及时买卖资产的风险；规模太小的基金有清盘流动性风险 |
| 纳斯达克100指数 | Nasdaq-100 Index | 美国纳斯达克交易所市值最大的100家非金融公司指数，以科技股为主 |
| 配置比例 | Allocation Ratio | 某只基金或某类资产在整体组合中的权重百分比 |
| 平准基金 | Stabilization Fund | 政府为稳定股市、减少波动而设立的专项基金；通常在市场大跌时入市托底 |
| 申购 | Subscription / Purchase | 向基金公司提交买入申请，按当日收盘后公布的净值成交；第二工作日确认份额 |
| 市盈率（PE） | Price-to-Earnings Ratio | 股价除以每股收益；衡量市场对企业盈利的定价水平，高PE意味着估值贵 |
| 市净率（PB） | Price-to-Book Ratio | 股价除以每股净资产；衡量市场对企业账面价值的溢价程度 |
| 托管费 | Custodian Fee | 基金资产由商业银行等托管机构保管所收取的费用；通常0.05%~0.25%/年 |
| 万份收益 | Per-10,000-Share Return | 货币基金每持有10,000份额一天所获得的收益，是货币基金的常用收益表示方式 |
| 系统性风险 | Systematic Risk | 无法通过分散投资消除的风险，如经济危机、利率政策变动、地缘政治冲击等 |
| 信息比率（IR） | Information Ratio | 超额收益 ÷ 超额收益的标准差；衡量基金经理创造稳定超额收益的能力 |
| 盈亏平衡点 | Break-even Point | 投资者总收益恰好覆盖所有成本（包括手续费）时的投资回报率 |
| 再平衡 | Rebalancing | 将组合中各类资产的实际比例调整回目标比例的操作；通常每年执行一次 |
| 沪深300指数 | CSI 300 Index | 由沪深两市市值最大、流动性最好的300只股票组成的宽基指数，代表A股主流 |
| 中证500指数 | CSI 500 Index | 沪深两市中，市值排名在301~800位的中型公司指数；代表中盘成长风格 |
